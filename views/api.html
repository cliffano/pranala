Jika anda ingin menggunakan API berikut ini, harap hubungi kami di redaksi@prn.la . Anda akan kami hubungi jika ada perubahan-perubahan pada API kami.

/v0/pendekkan

Contoh

  • <%= appUrl %>/v0/pendekkan?panjang=http%3A%2F%2Fberitakita.co.id%2Fterkini%2Findonesia-akhirnya-juara-piala-dunia-2058&format=json

Parameter

  • panjang - pranala panjang yang akan dipendekkan
  • format - format respons, misalnya: json, teks

Respons

  • json
    Sukses
    {
  "status":"sukses",
  "pendek":"<%= appUrl %>/g0L"
}
    Gagal
    {
  "status":"gagal",
  "pesan":"TIDAK_SAHIH"
}
  • teks
    Sukses
    <%= appUrl %>/g0L
    Gagal
